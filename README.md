# ATM API

Autor: Pedro Sicilia

Prueba técnica - API para operaciones en cajeros automáticos usando **Node.js** + **NestJS** + **TypeScript**.

##  Requisitos
- Node.js >= 18
- npm >= 9
- Git
- (Opcional) Docker

##  Instalación y ejecución
```bash
# Clonar el repositorio
git clone https://github.com/pedrosici/atm-api.git
cd atm-api

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run start:dev
